# mycourse
Автоматизированная Система «АРК» (АС «АРК»)

    Курсовая работа по дисциплине «Технология разработки программного обеспечения»
    Автор: Прохоров Данил Алексеевич (студент группы 24ИСП-1)

    Учебное заведение: Орский гуманитарно-технологический институт (филиал) ОГУ, факультет СПО

    Год выполнения: 2026

АС «АРК» — это оконное Windows-приложение, предназначенное для автоматизации базовых космологических расчётов. Программа ориентирована на студентов, преподавателей и всех, кто изучает основы астрофизики.

Цель разработки — ускорить и упростить выполнение типовых учебных задач, минимизировать ошибки ручных вычислений и обеспечить наглядную визуализацию результатов.
Функциональные возможности

    Расчёт лучевой скорости галактики по известному расстоянию.

    Определение расстояния до космологического объекта по заданной скорости удаления.

    Комплексный расчёт параметров объекта на основе красного смещения z.

    Оценка возраста Вселенной через постоянную Хаббла с автоматическим переводом в миллиарды лет.

    Конвертация единиц измерения расстояния: * Мегапарсеки (Мпк) в километры (км) и обратно.

        Мегапарсеки (Мпк) в световые годы (св. лет) и обратно.

    Построение диаграммы Хаббла — интерактивный график зависимости скорости удаления галактик от расстояния для визуального анализа данных.

    Валидация ввода — встроенная защита от ввода букв, отрицательных чисел и пустых полей с выводом предупреждающих сообщений.

Используемый математический аппарат

Приложение производит вычисления на основе следующих фундаментальных астрофизических зависимостей:

    Закон Хаббла:
    v=H0​⋅D

    Расчёт через красное смещение (z):
    v=z⋅c
    D=H0​v​

    Оценка возраста Вселенной:
    T≈H0​1​

    Константа: В расчётах с красным смещением используется константная скорость света c=3⋅105 км/с.

Инструкция по использованию
1. Главное меню

При запусти приложения открывается стартовое окно MainForm с перечнем доступных задач.

    Выберите переключатель (радиокнопку) напротив нужного режима:

        Расчёт скорости по расстоянию

        Расчёт расстояния по скорости

        Расчёт по красному смещению z

        Определение возраста Вселенной

        Конвертация единиц измерения

        Построение графика (диаграммы Хаббла)

    Нажмите кнопку «Выбрать» для перехода в рабочую форму.

2. Выполнение расчётов

Каждое рабочее окно имеет унифицированный интерфейс:

    Поля ввода для исходных данных (например, постоянной Хаббла H0​ и расстояния D).

    Кнопка «Рассчитать» — выводит результат (скорость удаления в км/с, а также дублирует расстояние в километрах и световых годах).

    Кнопка «Очистить» — мгновенно сбрасывает все текущие значения в форме.

    Кнопка «Назад» — возвращает пользователя в главное меню.

3. Построение диаграммы Хаббла

    Перейдите в форму графика из главного меню.

    Вручную добавьте несколько точек, введя для каждой расстояние и соответствующую скорость.

    Нажмите кнопку «Построить график». Точки отобразятся на координатной плоскости (ось X — расстояние в Мпк, ось Y — скорость в км/с).

    Для удаления всех точек нажмите «Очистить».

Обработка ошибок: Если вы ввели текст, оставили поле пустым или указали отрицательное число, система заблокирует расчёт и выведет подсказку. Исправьте значение и нажмите «Рассчитать» повторно.
Структура проекта (для разработчиков)

Проект разработан на языке C# в среде Visual Studio на базе платформы .NET 6.0 / 8.0.

    Form1 (MainForm) — центральный хаб и главное меню приложения.

    Form2 – Form6 — специализированные диалоговые формы для изолированного выполнения вычислительных задач и построения графиков.

    Валидация и фильтрация: Реализована через статические методы проверки (TryParsePositive, TryParseNonNeg) и перехват ввода на уровне компонентов (TextBox_KeyPress).

    Графика: Визуализация диаграммы Хаббла построена с использованием стандартного компонента DataVisualization.Charting.
