# mycourse
О программе
АС «АРК» представляет собой оконное Windows-приложение, разработанное в рамках курсовой работы по дисциплине «Технология разработки программного обеспечения».
Программа предназначена для автоматизации базовых космологических расчётов, основанных на законе Хаббла, эффекте Доплера и оценке возраста Вселенной. Она ориентирована на студентов, преподавателей и всех, кто изучает основы астрофизики.
Цель разработки – ускорить и упростить выполнение типовых учебных задач, минимизировать ошибки ручных вычислений и обеспечить наглядную визуализацию результатов.
Функциональные возможности
Расчёт лучевой скорости галактики по известному расстоянию (закон Хаббла: v=H_0?D).
Определение расстояния до космологического объекта по заданной скорости удаления.
Комплексный расчёт параметров объекта на основе красного смещения z(скорость v=z?cи расстояние D=v/H_0).
Оценка возраста Вселенной через постоянную Хаббла (T?1/H_0) с переводом в миллиарды лет.
Конвертация единиц измерения расстояния: мегапарсеки ? километры, мегапарсеки ? световые годы.
Построение диаграммы Хаббла – график зависимости скорости удаления галактик от расстояния (для визуального анализа).
Валидация ввода – защита от ввода букв, отрицательных чисел и пустых полей с выдачей информативных сообщений.
Системные требования
Операционная система: Windows 10 / 11 (64-разрядная)
Процессор: двухъядерный, от 2 ГГц
Оперативная память: не менее 4 ГБ
Свободное место на диске: ~100 МБ
Дополнительно: установленный пакет .NET Desktop Runtime 6.0 или более новая версия (8.0)
Инструкция по использованию
Главное меню
При запуске открывается главное окно с перечнем доступных задач в виде радиокнопок.
Выберите нужный расчёт:
Расчёт скорости по расстоянию
Расчёт расстояния по скорости
Расчёт по красному смещению z
Определение возраста Вселенной
Конвертация единиц измерения
Построение графика (диаграммы Хаббла)
Нажмите кнопку «Выбрать» – откроется соответствующая рабочая форма.
Рабочее окно задачи
Каждая задача имеет схожий интерфейс:
Поле(я) для ввода исходных данных.
Кнопка «Рассчитать».
Поле(я) для вывода результатов.
Кнопка «Очистить» для сброса значений.
Кнопка «Назад» для возврата в главное меню.
Пример выполнения расчёта:
Введите постоянную Хаббла H_0(км/с на Мпк) и расстояние (Мпк).
Нажмите «Рассчитать».
В поле результата появится скорость удаления в км/с, а также автоматически рассчитанное расстояние в световых годах и километрах.
Для задач с красным смещением дополнительно используется скорость света c=3?10^5км/с.
Построение диаграммы Хаббла
Форма графика позволяет:
Добавить несколько точек вручную (расстояние и соответствующая скорость).
Нажать «Построить график» – точки отобразятся на координатной плоскости (ось X – расстояние в Мпк, ось Y – скорость в км/с).
Очистить точки кнопкой «Очистить».
Обработка ошибок
При вводе текста или пустого поля появится сообщение с требованием ввести число.
При вводе отрицательного значения программа предупредит, что параметры должны быть положительными.
После исправления нажмите «Рассчитать» повторно.
Структура проекта (для разработчиков)
Решение Visual Studio включает несколько форм:
Form1 (MainForm) – главное меню.
Form2, Form3, …, Form6 – формы для конкретных вычислительных задач.
Вспомогательные статические методы валидации (TryParsePositive, TryParseNonNeg) и фильтрации нажатий клавиш (TextBox_KeyPress).
Весь математический аппарат реализован на языке C# с использованием платформы .NET 6.0/8.0, графики построены на базе компонента DataVisualization.Charting.
